# 脚本工具集

本目录包含 awesome-ai-ideas 项目的各类脚本工具。

## 目录结构

- **install/** - 安装脚本
  - Docker、CLI工具、高级技能安装器
- **config/** - 配置脚本
  - 项目配置、优化器
- **docker/** - Docker相关脚本
  - Docker安装、项目管理
- **utils/** - 工具脚本
  - 环境检查、验证工具

## 使用方法

所有脚本均在项目根目录执行：
```bash
bash scripts/install/docker_installer.sh
bash scripts/config/config_optimizer.sh
```
