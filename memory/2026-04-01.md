### [21:01] PowerShell + GitHub API经验教训
- `--jq` 在PowerShell中有编码问题，用 `--json` 或 `--template`
- `gh repo fork --list` 不存在，用 `gh api` 检查
- PowerShell `-f` 格式化JSON时：`{{` `}}` 转义为 `{` `}`，无占位符时用 `-f @()`
- `Set-Content -Encoding UTF8` 添加BOM，用 `[System.IO.File]::WriteAllText()` 代替
- 创建PR需要Git环境（`gh pr create`依赖git），无Git时用API创建
